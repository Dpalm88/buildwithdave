## 🚀 The Mission
Helping local Portland businesses move away from bloated, expensive hosting and into modern, secure, and lightning-fast cloud environments.

## 🛠 Infrastructure & DevOps
This project isn't just a website; it's a demonstration of modern web infrastructure:
* **Edge Hosting:** Deployed on **Cloudflare Pages** for global distribution and 100/100 Lighthouse performance scores.
* **CI/CD Pipeline:** Fully automated deployment workflow. Every `git push` triggers a fresh build and edge-cache purge.
* **DNS Management:** Migrated from legacy providers to Cloudflare for advanced security (WAF), Brotli compression, and SSL/TLS.
* **Local SEO:** Optimized with schema markup and mobile-responsive design for the Portland metro market.

## 📁 Project Highlights
* **Ridgeline Junk Removal:** A full infrastructure migration from Netlify/Namecheap to Cloudflare, including custom domain and iCloud Mail integration.
* **Strava Dashboard:** Integrated API project demonstrating data visualization and personal performance tracking.
